#### 准备

`npm install -g webpack`

`cd /path/to/project`

`npm install`

如果使用compass，需先安装compass：

`gem install compass`

#### 开发环境

`webpack`

或

`webpack -d`

或

`webpack -development`

实时监听：`webpack --watch`

本地服务器：`npm run server`。服务器运行在`http://127.0.0.1:8080/`

### 生产环境

`webpack -p`

或

`webpack --production`
